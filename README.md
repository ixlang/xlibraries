# xlibrarys
xlang的基础库 

## FileSystem
### 文件系统相关操作的包， File 对象

## FileStream 
### xlang 文件读写流对象， FileInputStream 和 FileOutputStream

## Runtime 
### 运行时相关的包， 目前只有进程相关操作， 并且只支持WINDOWS和LINUX

## XPlatform
### 平台基础相关的包 ， 主要用于解决WINDOWS字符集的问题

## QT&XLIB
### 基于QT的UI库 XLANG 层封装， 部分源文件引用自XStudio项目， 

## QXLibrary 
### QT的C++层项目
